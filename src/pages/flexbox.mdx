> React Native中flex布局和CSS flex布局基本一致，**如果熟悉CSS flex布局，可以跳过本章**。

组件使用**flexbox**算法指定它们子组件的布局。使用flexbox让你指定布局，扩展或收缩以适应各种屏幕的尺寸。

你能无缝地混用flex布局和像`width: 100`这样的固定布局，以适应各种布局。

为了给组件的子组件选择正确的布局，下面三项你需要做些选择：

| 属性         | 默认值      | 选项                                                     | 描述                                                                                                                                                                                                                                              |
| ---------------- | ------------ | ----------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `flexDirection`  | `column`     | `row, column`                                               | 你想要垂直布局(`column`)还是水平布局(`row`)？这个选择决定了你布局的**主轴**方向。                                                                                                                                                                               |
| `justifyContent` | `flex-start` | `flex-start, center, flex-end, space-around, space-between` | 内容是如何沿着主轴分布的？ 沿主轴方面的开始，中间，末尾，还是两端均匀分布？                                                                                                  |
| `alignItems`     | `stretch`    | `flex-start, center, flex-end, stretch`                     | 内容如何沿着附轴分布？ |
