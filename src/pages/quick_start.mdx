在本章节，我们将会使用`expo-cli`来创建一个新的React Native应用。

你需要安装Node.js （v10及以上版本），如果你还没装的话，到[nodejs.org](https://nodejs.org/en/download/)下载安装程序，选择你电脑平台对应的安装包，跟着教程安装就行。

## 安装

首先，安装[`expo-cli`](https://docs.expo.io/versions/latest/workflow/expo-cli/)

```bash
npm install -g expo-cli
```

使用expo-cli创建一个全新的应用：

```bash
expo init MyApp
cd MyApp
npm start
```

`expo-cli`下载依赖会花费几分钟。依赖装完后，你应该能看到像下面这样的二维码。
