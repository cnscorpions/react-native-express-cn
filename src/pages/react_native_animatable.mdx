React Native Animatable是一个声明式动画库。参见Github repo [这里](https://github.com/oblador/react-native-animatable)。

创建动画比使用`Animated`要容易得多。与`Animated`一样，我们首先需要使用一个专门封装的组件：`Animatable.View`、`Animatable.Text`、`Animatable.Image`或`Animatable.createAnimatableComponent(Component)`。从那里，我们添加一个`animation` prop 到我们的组件中，以指定使用哪种动画，例如`fadeIn`。我们可以通过prop来控制动画的参数。

# 例子
