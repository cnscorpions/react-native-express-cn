
React Native提供了两个开箱即用的动画API，`Animated`和`LayoutAnimation`。`Animated`的使用频率要高得多，所以我们只在这里介绍一下它。如果你对`LayoutAnimation`很好奇，你可以在[docs](https://facebook.github.io/react-native/docs/animations.html#layoutanimation)中阅读更多。

`Animated`是一个有点命令式API的意思，这意味着你将手动指定何时开始和停止动画。这给了你很多控制权，但很快就会变得乏味。考虑使用[React Native Animatable](react_native_animatable)来实现围绕`Animated`的基于组件的声明式API。

如果你计划构建一个丰富的交互式UI，你还需要`PanResponder`API来跟踪和响应触摸事件。
