
自ES5于2009最初发布，ES2015，或ECMAScript 2015是Javascript语言的首个重大更新。你经常会看到ES2015被称作为它原本的名字，ES6，因为它是ECMAScript的第6个版本。

在大多情况下，所有的ES2015特性被现代游览器支持。然后，使用Babel让我们能使用未被支持的新特性，确保JS运行在更过的平台上。
For the most part, all ES2015 features are now available in modern JavaScript engines. Using Babel, however, gives us access to newer features which aren't yet, while ensuring our JavaScript runs on even more platforms. 

React Native使用Babel来启用ES2015和更新的语言特性，确保跨平台一致性，所以你的JS代码能运行在Android, iOS, Windows和其他平台。
React Native uses Babel to enable ES2015 (and newer) features and ensure cross-platform consistency, so your JavaScript can run on Android, iOS, Windows, and other platforms.
